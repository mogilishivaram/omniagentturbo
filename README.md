## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create `.env.local`:
   ```
   OPENAI_API_KEY=your-key
   ```

3. Run the app:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000/OmniAgent](http://localhost:3000/OmniAgent)

## Notes

- Currently uses `gpt-4-turbo`
- You can enable `gpt-4o` in `gpt.js` when it's released in API
